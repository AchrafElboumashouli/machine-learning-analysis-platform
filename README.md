# 🧠 DataMind AI

### Machine Learning Data Analysis Platform

🚀 A full-stack AI-powered platform that allows users to upload datasets, analyze data, train machine learning models, and visualize results — all in an intuitive step-by-step workflow.

---

## ✨ Key Features

* 📂 Upload datasets (CSV, Excel, JSON)
* 📊 Perform exploratory data analysis (EDA)
* ⚙️ Configure preprocessing pipelines
* 🤖 Train ML models (Classification, Regression, Clustering)
* 📈 Visualize results with interactive charts
* 📤 Export predictions, models, and reports

---

## 🧠 Supported Machine Learning Tasks

* **Classification** → Predict categories
* **Regression** → Predict numerical values
* **Clustering** → Discover hidden patterns

---

## 🏗️ Architecture

```
Frontend (Next.js)
        ↓
Backend API (Node.js + Python)
        ↓
Machine Learning Models (scikit-learn, TensorFlow)
```

---

## 🛠️ Tech Stack

### Frontend

* Next.js 16
* React 19
* Tailwind CSS
* shadcn/ui
* Recharts

### Backend

* Node.js (API Routes)
* Python (scikit-learn, Pandas, NumPy)
